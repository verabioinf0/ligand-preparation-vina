# Подготовка лигандов для AutoDock Vina

Тестовое задание на стажировку.

Jupyter-ноутбук для автоматической конвертации большого количества малых молекул из форматов SDF и SMILES в формат PDBQT, необходимый для молекулярного докинга в AutoDock Vina.

## Что реализовано

- Чтение молекул из SDF-файлов (включая 2D-структуры и молекулы без водородов)
- Чтение молекул из SMI-файлов
- Добавление водородов и генерация 3D-конформеров через RDKit (ETKDGv3 + оптимизация UFF)
- Конвертация в PDBQT через библиотеку Meeko
- Сохранение по одному файлу на лиганд, имя файла берётся из исходного
- Обработка ошибок и статистика по каждому источнику
- **Дополнительное задание:** алифатические циклы с числом атомов 6 и более помечаются как гибкие (`min_ring_size=6` в Meeko)

## Структура репозитория

```
ligand-preparation-vina/
├── ligand_preparation.ipynb   # основной ноутбук
├── input/                     # входные файлы
│   ├── example.sdf
│   └── example.smi
└── output/                    # сюда сохраняются PDBQT-файлы
```

## Установка зависимостей

```bash
pip install rdkit meeko
```

## Использование

Открыть `ligand_preparation.ipynb` в Jupyter и запустить все ячейки (`Run All`).

Входные файлы должны находиться в папке `input/`. Результаты сохраняются в `output/`.

## Стек

- Python 3.9+
- RDKit — чтение молекул, генерация 3D-конформеров
- Meeko — подготовка к докингу, запись PDBQT
